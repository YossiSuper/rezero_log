現在準備中です。

概要

リゼロファンによる、リゼロファンのためのリーディングログサイトです。リゼロ初心者の方でも、「楽しむにはまず何をみれ(読め)ば良いのか。」をわかりやすくまとめつつ、自分がどこまでみたのか(読んだのか)を整理できるような機能を提供するのが目的です。

このサイトはあくまでファンによるファン活動の一環であり、商用的な目的は一切ありません。そのため、サイト内にアフィリエイトリンクを掲載することは一切ないものとします。

使う ->

リゼロの小説やアニメの関係を一目で確認できます。

このサイトの概要とマイルストーンを掲載しています。
